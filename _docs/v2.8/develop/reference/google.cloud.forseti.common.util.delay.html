google.cloud.forseti.common.util.delay module¶

Tool used to delay initial execution of a function.

delay(delay_by, clock=None)[source]¶

Delays execution of the decorated function.

Parameters:
  • delay_by (int) – Number of seconds to delay by.
  • clock (time.sleep) – Clock used to measure delay (mainly used for tests). If none is provided clock will be created.
Returns:

Wrapped function.
Return type:

function
